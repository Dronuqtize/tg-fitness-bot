Fitness Bot

Быстрый старт и команды. Открывай внутри Telegram.

Запуск

  1. Открой чат с ботом
  2. Нажми кнопку Mini App
  3. Проверь команды и примеры

Основные команды

  • /today — план на сегодня
  • /progress — вес и обмеры
  • /calendar — календарь
  • /attendance — табель
  • /chart — график
  • /advice — ИИ‑совет
  • /pdf — PDF отчёт

Напоминания

Примеры:

/reminder set water 10:00
/reminder set ai 21:00
/reminder off water
